1. Temat strony: Zespół Gospodarki Komunalnej

2. Sekcje strony:
    - Nagłówek (header): Logo, nazwa
    - Treść główna (main): Główna część strony
    - O nas (section): Inforamcje o firmie
    - Oferta (section): Co oferuje firma
    - Kontakt (section): Formularz kontaktowy
    - Stopka (footer): Szybki kontakt

3. Grafiki:
    - Logo firmy
    - Pracownik
    - Budynek (godziny pracy)

4. Funkcjonalności:
    - Interaktywny formularz kontaktowy
